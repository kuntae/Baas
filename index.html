이 프로젝트는 2013년도 한성대학교 컴퓨터공학과 Baas팀의 Bacas 프로젝트 repository 입니다.

$ git clone https://github.com/kuntae/Baas.git

What is Baas?

Baas는 Backend as a Service의 줄임말 입니다. 어플리케이션 개발자들을 서비스 대상으로 하여 Backend기능을 지원하는 서비스입니다. 대표적으로 데이터베이스를 이용한 다양한 기능들을 구현하고 서비스 사용자들에게 제공하는 것입니다. 지원하는 기능의 예는 어플리케이션 사용자의 정보, 위치정보, 메시지전송, 사용빈도관리, 파일관리 같은 기능들이 있습니다.

What is Bacas?

저희가 만든 Baas의 이름으로 Backend as 'cool' a service의 줄임말 입니다. 멋진 서비스를 만들겠다는 뜻으로 'cool'이라는 단어를 추가했습니다. 저희 Bacas는 Node.js와 mongodb를 사용하여 만들었습니다.

Bacas는 어플리케이션 개발자들이 가장 선호하는 기능으로 조사된 기능을 바탕으로 만들었습니다. 사용자관리, 위치정보, 메시지 전송, DB구조, 사용선호도, SNS등의 기능을 지원하며 Bacas SDK를 구현하여 안드로이드 앱 개발자들에게 지원하고 있습니다.

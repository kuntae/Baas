Welcome to Bacas Pages.

이 프로젝트는 2013년도 한성대학교 컴퓨터공학과 Baas팀의 Bacas 프로젝트 repository 입니다.

이 프로젝트를 다운 받기 위해서는 git clone 명령어를 통해 다운받으시거나 압축된 형태를 다운로드 하실 수 있습니다.

$ cd your_dir
$ git clone https://github.com/kuntae/Baas.git

What is Baas?

Baas는 Backend as a Service의 줄임말 입니다. 어플리케이션 개발자들을 서비스 대상으로 하여 데이터베이스를 이용한 다양한 기능들을 구현하 서비스 사용자들에게 제공하는 서비스를 말합니다. 지원하는 기능은 어플리케이션 사용자의 정보, 위치정보, 메시지전송, 사용빈도관리, 파일관리 같은 기능들이 있습니다.
